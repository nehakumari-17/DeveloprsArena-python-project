# Sales Data Analysis Report

## Project Overview

This project analyzes sales data using Python and the Pandas library. The objective is to explore the dataset, calculate key sales metrics, and generate useful business insights from the data.

## Dataset Information

The dataset was loaded from `sales_data.csv` and contains sales-related records used for analysis.

## Analysis Performed

1. Loaded the dataset using Pandas.
2. Displayed the first 5 rows of the dataset.
3. Checked dataset information including rows, columns, and data types.
4. Calculated total sales.
5. Calculated average sales.
6. Identified the highest sale value.
7. Identified the lowest sale value.
8. Generated summary insights from the sales data.

## Findings

The analysis provides useful insights into sales performance and helps understand overall business trends. Key metrics such as total sales, average sales, highest sale, and lowest sale were successfully calculated.

## Tools Used

* Python 3
* Pandas
* Visual Studio Code (VS Code)
* Git and GitHub

## Conclusion

This project demonstrates basic data analysis using Python and Pandas. It shows how datasets can be loaded, explored, and analyzed to obtain meaningful business insights.
